Instructions

NOTE: This is an abridged version of the assignment instructions. Refer to the PDF posted on Blackboard for the complete version of the assignment instructions.

Upload the sql for the following queries in a ZIP archive. The file names should be for the form "N.sql", where N is the respective query number.

Queries
  • 1. Write a query that describes the routes table. You should use the DESCRIBE command in MySQL.
  • 2. Write a query that outputs the route_id, route_short_name, and route_long_name columns from the routes table. Order by route_id and take only the first 10 results.
  • 3. Write a query that outputs all columns (*) in the stops tables where the stop_name contains (LIKE) “Collins Circle”.
  • 4. Make a copy of Query 3 and modify it to now include the all columns (*) from the stop_times table, as well. This query should produce all columns from both the stops and stop_times tables where the stop is Collins Circle. Refer to the documentation for which column to use to make the join.
  • 5. Make a copy of Query 4 and add to the WHERE clause a condition that the bus should arrive at Collins Circle at or before 5:20 PM.
  • 6. Make a copy of Query 5 and modify it to include data from the routes and trips tables. Show all columns (*). Use the join columns listed in the GTFS documentation. In the WHERE clause, add conditions that the route_short_name is the 114, and that the trip_headsign is “West”.
  • 7. Make a copy of Query 6 and modify it to include the information in the calendar table. Make sure that the trip in in service the day of the final exam. (Note that CDTA does not provide an end_date, so just use start_dates.) Show all columns.
  • 8. Make a copy of Query 7 and modify to output only the trip_id of the trip that arrives at Collins Circle as close to 5:20 PM as possible, without arriving after that time.
  • 9. Write a new query that outputs all the stop_names and arrival_times for the trip that we identified in Query 8. Make a copy of Query 8 and use it in a subquery to make sure the trip is the one we identified in Query 8.
  • 10. Make a copy of Q9 and modify it to get the stop names and arrival times of the 114 bus that arrives at Collins Circle as close to 7:50 PM as possible (not earlier), then heads East into Winter Vacation.
