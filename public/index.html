SQL Scripts Uploader

Instructions

Upload the sql for the following queries in a ZIP archive. The archive structure must be flat: i.e. when the archive is extracted, the sql files should not be in a directory. The file names should be for the form "N.sql", where N is the respective query number.

Queries
  • 1. Display all the data in the Department table.
  • 2. Display the flast, ffirst, fdept, and fsalary columns from the Faculty table.
  • 3. Display all the first names in the Faculty table and remove duplicate rows.
  • 4. Display all the data in the Course table in descending cname order.
  • 5. Display the id, full name, and salary columns from the Faculty table and sort the output in telephone book order.
  • 6. Join the Department and Faculty tables and display all the columns.
  • 7. Join the Faculty and Course tables.
  • 8. Join the Department and Course tables.
  • 9. Join the Course and Section tables.
  • 10. Do a 3-way join of the Department, Course, and Section tables.
  • 11. Do a 3-way join of the Department, Faculty, and Section tables.
  • 13. Find departments that offer courses.
  • 18. Find departments that don’t offer courses.
  • 19. Find courses that have sections.
  • 20. Find courses that have sections.
  • 25. Join subordinates and managers… they both just happen to be in the same table: Faculty.
  • 26. Find all the subordinates who make more than their managers.
  • 27. List all the departments in Faculty and Course.
  • 28. Count the number of rows in Faculty.
  • 29. Count the number of non-null fid values in Faculty.
  • 30. Count the number of departments that offer courses.
